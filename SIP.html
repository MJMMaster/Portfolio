Project Description

Split Shift is a single-player prototype developed in Unreal Engine that explores a dual-character gameplay system where two separate worlds continue running even when the player is not controlling one of them. One character is a tech intern navigating suspicion and social interactions in an office environment, while the other is a combat-focused hero inside a simulation. The project focuses on designing UI systems that communicate information between these worlds, allowing players to strategically decide when to switch characters based on the status and risks occurring in the other world.

Innovation Claim

This project is innovative because it creates a dual-character system where both worlds continue running in real time and players rely on cross-world UI feedback to determine when to switch between characters. The innovation comes from making shared information and strategic switching the core gameplay mechanic.
